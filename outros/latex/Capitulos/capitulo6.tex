\chapter{Considerações finais}
Conforme demonstrado nesta proposta, o desenvolvimento do sistema \textit{web} para auxiliar na criação, monetização e divulgação de conteúdos dinâmicos com ênfase em viralizações, torna-se uma importante ferramenta para criadores de conteúdos presentes em redes sociais.  

Com testes a serem executados no \textit{website} www.otariano.com pretende-se validar a plataforma desenvolvida como uma ferramenta para criadores de conteúdos. O site em questão implementa, de forma rudimentar, as funcionalidades propostas por esse site através de plugins personalizados para o  \textit{WordPress}\footnote{https://wordpress.org/}. Com este trabalho, busca-se a consolidação de uma plataforma escalável para possibilitar a monetização de conteúdos a serem publicados em redes sociais.
